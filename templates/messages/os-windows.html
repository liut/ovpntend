操作系统: Windows

首先要安装 OpenVPN 客户端, 下载地址 戳这里. 根据自己的系统是 Windows XP 还是 Windows Vista/7/8/10, 以及系统是 32 位还是 64 位, 来确定下载哪一个客户端. (如何确定计算机运行的是 32 位还是 64 位版本的 Windows 操作系统)

下载完成后，安装 OpenVPN 客户端，全部用默认选项 (一路 Next / 下一步). 然后按照如下步骤配置.

  1. 在桌面的 OpenVPN 图标上点击右键 - "属性"

  2. 在属性的 "快捷方式" 选项卡下点 "打开文件位置" 按钮

  3. 在资源管理器的地址栏退到上一级名为 "OpenVPN" 的文件夹

  4. 打开 "OpenVPN" 文件夹下的 "config" 文件夹

  5. 将邮件附件中的证书下载到本机, 不要打开, 直接移动到 "config" 文件夹中

  6. 回到桌面, 在 OpenVPN 图标上点击右键 - "以管理员身份运行"

  7. 此时右下角托盘区应该已经出现带锁的 OpenVPN 图标 (如果没看到可以点小三角形展开托盘看看), 右键点击图标，选择 "Connect"

  8. OpenVPN 会打开一个窗口提示连接状态, 等待这个窗口自动消失, 同时右下角带锁图标变绿, 说明已经连接成功. 此时网络环境为直连公司内网, 可以访问代码仓库、管理员后台等内部系统.
