注意事项

OpenVPN 证书可用于接入公司内网, 请勿外泄. 如果不慎外泄或安装了证书的电脑失窃, 请联系技术团队销毁旧证书并重新签发.

安装方法

操作系统: Linux

因为 Linux 发行版实在太多了, 无法一一说明, 还请移步官方文档吧.

_(:з」∠)_

常用发行版简要说明:

Debian 和 Ubuntu 系 使用 sudo apt-get install openvpn 安装

Redhat 和 CentOS 系 使用 sudo yum install epel-release 安装扩展, 然后 sudo yum install openvpn 安装openvpn

安装后将证书文件放在/etc/openvpn下

启动: systemctl start openvpn@配置文件名.service (Redhat 不能用该方法, 可以尝试systemctl start openvpn)

如有疑问, 请联系技术团队.
